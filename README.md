# ⏳ Python Countdown Timer

A simple command-line countdown timer built with Python. The program counts down from a user-specified number of seconds and displays the remaining time in **MM:SS** format until the timer reaches zero.

## 🚀 Features

* Accepts countdown time in seconds
* Displays time in **MM:SS** format
* Updates every second
* Prints a completion message when the timer ends
* Beginner-friendly and easy to understand

## 🛠️ Technologies Used

* Python 3
* Built-in `time` module
